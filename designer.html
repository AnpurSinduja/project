Project Manager

Agarwal

The project manager controls the workflow and ensures that each step of each project is completed within the budgets and schedules specified. This function is a combination of creative and operational in nature, since it requires a thorough understanding of the design process but is ultimately a business-oriented role. From the conceptual stage through the final installation, the project manager implements the right order of operations for each project.
